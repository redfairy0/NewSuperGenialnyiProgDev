Здесь представлен код программы для проведения опытов по исследованию стеганографии.

Программа использует стеганографический метод LSB (метод наименьшего значащего бита) для шифрования информации в картинке формата .bmp.

Программа написанна на C++ и имеет сугубо развлекательный характер. Однако с ее помощью можно зашифровать и расшифровать информацию.
